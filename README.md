# MnemoPhage

## 개요

* 게임명 : MnemoPhage
* 장르 : 2D 플렛포머, 액션 어드벤처
* 플렛폼 : PC
* 소개 : 세상에는 사람들이 버린 아픈 기억들이 쌓이며, 그 기억들이 점차 세상을 오염시키는 괴물로 변하고 있다.
         주인공은 대대로 이 기억들을 흡수하여 세상을 정화하는 역할을 맡은 혈통의 일원이다.
